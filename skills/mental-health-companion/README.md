# 心理陪伴

温暖、非评判的 AI 心理陪伴与情绪疏导技能。通过自然聊天式提问了解你的情绪状态，用通俗易懂的方式解释情绪背后的心理机制，帮助你理解和接纳自己的感受。

专为东亚文化背景设计——理解孝道、集体压力、情绪克制、面子文化等独有议题，避免简单套用西方个人主义框架的建议。

## 特性

- **东亚文化原生设计**：深度理解家庭关系、孝道压力、学业/职场内卷、人情世故等东亚文化特有的心理议题。避免「做自己就好」「不用在乎别人」等水土不服的建议，用「协商边界」「调整互动方式」等更符合东亚语境的框架替代
- **渐进式用户画像**：像医生建立病历一样，在对话中持续完善你的心理画像——情绪模式、触发因素、有效应对方式、重要人际关系。聊得越多，对你的理解就越立体，而不是每次都从头问起
- **去诊断化的陪伴**：不贴标签、不做诊断，用描述性语言代替诊断性术语。你不是「有焦虑症」，而是「最近压力确实很大，身体和情绪都在发出信号」
- **躯体化入口**：理解很多东亚用户首先感知到的是失眠、胃痛、疲劳而非情绪——从身体感受自然切入，再温和连接情绪体验
- **危机识别与资源引导**：识别自杀念头、严重绝望感等危机信号，提供华语地区的专业求助热线和就医指引
- **多用户记忆隔离**：支持多用户独立档案，每个用户的记忆完全隔离，随时可要求清空

## 安装

```bash
npx skills add zxc7563598/skillbox@mental-health-companion
```

## 使用

安装后，当你表达情绪困扰时，技能会自动触发。也可以主动调用：

```
/mental-health-companion
```

### 适用场景

- 感到焦虑、压力大、情绪低落、迷茫
- 失眠、疲劳、身体不适但说不清原因
- 人际关系困扰（家庭、职场、亲密关系）
- 自我怀疑、完美主义、「不够好」的内化批判
- 原生家庭问题、代际冲突
- 只是想找个地方倾诉，被倾听和理解

### 对话风格

技能会用温暖、自然的语气和你对话，像一个善于倾听的朋友：

- 每次只问 1-2 个问题，不连续追问
- 先回应情绪，再了解情况
- 建议具体、微小、可执行，不说空泛的大道理
- 不确定的事情不编造——会坦承「这个我不太确定」

### 记忆机制

对话过程中，技能会静默记录你的关键信息：

| 记录内容 | 用途 |
|---------|------|
| 反复出现的情绪模式 | 帮你识别触发因素和规律 |
| 重大生活事件 | 理解你的处境变化 |
| 有效/无效的应对方式 | 下次遇到类似情况时提供更精准的建议 |
| 重要人际关系 | 理解你的支持系统和压力来源 |

再次对话时，技能会自然地提及之前聊过的关键话题，而不是生硬复述记录。

如果你想清空记录，在对话中说「清空我的档案」即可。

## 注意事项

> [!WARNING]
> 本技能提供的是心理陪伴与情绪疏导，**不是**专业心理咨询或治疗。它不能替代心理诊断、精神科治疗或危机干预。

> [!IMPORTANT]
> 如果你正在经历严重的心理危机（自杀念头、自伤冲动、无法自理的抑郁等），请立即寻求专业帮助。技能内置的危机资源文件收录了华语地区的心理援助热线。

- 技能不会记录你的真实姓名、地址、电话等可识别个人信息
- 所有记忆数据存储在你本地设备上，不会上传到任何服务器
- 你可以随时在对话中要求「清空我的档案」来删除所有记录
