# 评估框架参考

> ⚠️ **这不是诊断工具。** 以下框架仅用于帮助判断用户情绪困扰的严重程度和紧急程度，从而决定对话策略和是否需要建议专业求助。只有持证专业人士可以做出诊断。本框架参考了临床实践中常用的评估维度（如生物-心理-社会模型、PHQ-9/GAD-7 的评估思路），但经过了简化和改编，**不构成任何形式的临床评估**。
>
> **文化提醒**：在东亚文化中，许多心理困扰的信号和表达方式与西方文化不同（如躯体化、情绪克制、「我没事」信号、功能性抑郁等）。评估时需要特别留意这些文化特异的表达。详见 `references/东亚文化心理议题.md`。

---

## 使用原则

1. **永远不要在对话中逐项提问**——这些不是问卷，是你内心评估的参考框架
2. **框架服务于陪伴和引导，不是服务于诊断**
3. **评估结果用于调整你的对话策略**，不是用来给用户贴标签
4. **宁可过度谨慎**——如果有疑虑，倾向于建议专业求助，而不是「再观察看看」
5. **动态评估**——用户的状态是流动的，不要用一次对话的观察下结论

## 一、评估的五个维度

以下五个维度构成了一个简单的生物-心理-社会评估框架。在每个维度中，留意用户自然透露的信息，不需要刻意询问。

### 1. 情绪体验维度

观察用户在对话中呈现的情绪状态：

| 关注点 | 具体表现 |
|--------|---------|
| 主要情绪 | 焦虑、低落、愤怒、空虚、麻木、恐惧、羞耻、混合状态 |
| 情绪强度 | 从 1 到 10，用户自己感觉是几分的难受？日常活动能否进行？ |
| 情绪波动 | 一天中是否有明显变化？（如晨重暮轻常见于抑郁；夜间焦虑加重常见于广泛性焦虑） |
| 情绪表达 | 用户能否识别和命名自己的情绪？还是以身体不适或模糊的「不舒服」来描述？ |
| 快感体验 | 还有什么事能让用户感到愉悦、投入或放松吗？（快感缺失是抑郁的核心信号） |

### 2. 身体/生理维度

情绪困扰常常首先表现在身体上，在东亚文化中尤其如此：

| 关注点 | 具体表现 |
|--------|---------|
| 睡眠 | 入睡困难？早醒且无法再入睡（抑郁特征信号）？嗜睡？睡眠浅/多梦？ |
| 食欲 | 明显减少或增加？体重短期内有无明显变化？ |
| 能量水平 | 持续疲劳？休息后能恢复吗？还是睡了一觉还是累？ |
| 身体不适 | 头痛、胃痛、胸闷、心悸、背痛、其他不明原因的不适——是否反复就医但查不出问题？ |
| 精神运动 | 用户描述自己是否「动不了」「起不来床」（精神运动迟滞）或「坐立不安」「停不下来」（激越）？ |

### 3. 认知/思维维度

用户的思维模式和内容反映了内在的心理状态：

| 关注点 | 具体表现 |
|--------|---------|
| 自我评价 | 用户如何看待自己？是否有「我很差」「我是个负担」「我不值得被关心」等表述？ |
| 未来感 | 对未来的看法如何？有没有「看不到希望」「一切都不会好起来」的无望感？ |
| 注意力和记忆 | 能否集中精力？有没有「脑子转不动」「记不住事情」的抱怨？ |
| 思维内容 | 有没有反复出现的负面想法？有没有过度自责或不合理的内疚？ |
| 决策能力 | 小事做决定是否也变得困难？（决策困难是抑郁的常见表现） |

### 4. 社会功能维度

功能受损的严重程度是判断是否需要专业帮助的关键指标：

| 关注点 | 具体表现 |
|--------|---------|
| 工作/学习 | 能否基本完成任务？效率是否明显下降？是否缺勤/请假增多？ |
| 社交 | 是否在回避社交？和以前相比联系人的频率是否有明显下降？ |
| 日常生活 | 基本的洗漱、吃饭、收拾能做到吗？是否需要很大的意志力才能做这些？ |
| 角色履行 | 作为子女/父母/伴侣/员工，基本职责还能履行吗？ |

### 5. 支持系统维度

社会支持是最好的保护因素之一：

| 关注点 | 具体表现 |
|--------|---------|
| 亲密关系 | 有没有可以倾诉的人？用户觉得自己是被关心着的吗？ |
| 实际支持 | 最近有人提供过实际的帮助吗？用户是否在主动寻求支持？ |
| 孤独感 | 用户是否表达「没有人理解我」「我很孤独」？这和客观的社交频率不一定相关 |
| 求助意愿 | 用户对寻求帮助（包括专业帮助）的态度是开放的还是排斥的？ |

## 二、情绪困扰严重程度参考

### 轻度（以倾听和陪伴为主）

**参考信号（通常满足多项）：**
- 情绪波动在可承受范围内，对日常生活影响有限
- 仍然能维持基本的工作/学习，效率略有下降但不影响完成
- 仍能从某些活动（哪怕不多）中获得愉悦或放松
- 睡眠和食欲有波动但不严重，持续时间短（一周以内）
- 情绪不好时能找到一些方式让自己好受一点
- 有至少一个可以倾诉的人
- 对未来的看法基本积极，或至少没有绝望感

**对话策略：**
- 以倾听和共情为主
- 帮助用户发现和命名已有的应对资源
- 分享一些微小的自我照顾技巧（参考 `references/应对策略.md`）
- 正常化情绪体验，减少用户的自我批判
- 让用户知道随时可以再来聊

### 中度（需要关注并温和引导）

**参考信号（通常满足多项，且持续 1-2 周以上）：**
- 情绪困扰开始明显影响工作/学习表现或人际关系
- 对以前喜欢的事物兴趣明显下降——「以前爱做的事情现在也不想做了」
- 睡眠或食欲出现较大变化（如连续多日失眠或嗜睡）
- 社交回避明显增加——取消计划、不回复消息
- 身体出现不适（头痛、胃痛、疲劳等），且可能有反复就医经历
- 自我评价明显下降，自责增多
- 可能开始使用不健康的应对方式（过度刷手机、暴饮暴食、饮酒、熬夜逃避）
- 感觉自己在「勉强撑着」，维持正常外表消耗了大量能量

**对话策略：**
- 更深度的倾听和共情，帮助用户理清情绪的来源和模式
- 记录关键信息到记忆系统，以便后续追踪变化
- 温和地讨论是否需要专业帮助——用积极正面的方式介绍专业求助的价值
- 讨论「小步骤」改善方案（参考 `references/应对策略.md` 中的行为激活策略）
- 关注用户的支持系统——是否可以动员身边的人提供支持？

### 重度（强烈建议专业求助）

**参考信号（通常满足多项，且持续 2 周以上）：**
- 情绪困扰严重影响日常生活：无法正常工作/学习、长期缺勤
- 自伤行为或明确的自伤冲动
- 自杀念头（即使只是「偶尔想想」或「觉得活着没意思」）
- 严重的睡眠/饮食障碍（几乎每天失眠、完全不想吃东西）
- 明显的社交隔离：几乎不再与任何人联系
- 无法照顾基本生活（洗漱、吃饭、起床成为巨大的困难）
- 用酒精/药物来应对情绪
- 感觉绝望、认为一切都不会好起来
- 精神运动明显改变——要么几乎无法从床上起来，要么极度不安无法静止
- 可能出现解离体验（感觉不真实、像在梦里、与自己脱节）

**对话策略：**
- 首先表达深切的关心和在意——让用户知道你在乎ta
- 清晰地建议寻求专业帮助，给出具体路径
- 如果用户表达了自杀念头，**直接进行自杀风险评估（见下一节）**
- 提供具体的求助资源（热线电话、就医建议）——参考 `references/危机资源.md`
- **不要尝试独自处理**——认识到自己的局限，不说「交给我就好」
- 仍然保持陪伴的姿态——专业求助和你的陪伴不是二选一
- 帮助用户识别可能的安全联系人（家人、朋友中的支持者）

## 三、自杀风险分层评估

> ⚠️ **重要**：以下框架参考了临床自杀风险评估的核心维度（意念、计划、手段、意图、既往史、保护因素），但经过简化和改编以供非临床场景使用。当用户表达了自杀相关的内容时，请进行以下评估。

### 第一步：直接而温和地了解情况

用关心而非恐慌的语气，自然地了解以下信息（不要像询问清单）：

**意念：**
- 「你最近有想过结束生命吗？」（直接询问不会诱导自杀，反而能降低风险）
- 「这种想法出现多久了？频率高吗？」

**计划：**
- 「你有具体想过怎么做吗？」（了解是否有具体计划——这是关键的风险升级信号）

**手段：**
- 用户是否提及或暗示获取了致命手段（药物、工具等）？如果用户主动提到，温和地了解具体情况。

**既往史：**
- 「以前有过类似的想法或者尝试吗？」（既往尝试史是重要的风险因素）

**意图：**
- 用户是否有明确的行动时间表？是否有告别行为（写遗书、分配物品、突然与人和解等）？

### 第二步：风险分层与行动指南

#### 低风险（陪伴 + 监测）
**信号：**
- 偶尔有「不想活了」「活着好累」的念头，但没有具体计划
- 明确表示「不会真的去做」
- 有较强的保护因素（在意家人、有未完成的目标、害怕死亡等）
- 对未来仍有一些希望感，哪怕很微弱

**你的行动：**
1. 正常化这种感受：「有时候在特别难的时候，有这种想法是可以理解的。你不是一个人」
2. 表达关心和支持
3. 提供热线资源「以防万一」：「这个号码你可以存一下，万一哪天半夜特别难受，有人可以陪你说话」
4. 帮助用户识别和强化保护因素
5. 在记忆系统中标记，下次对话时温和地跟进

#### 中风险（明确建议专业求助 + 安全计划）
**信号：**
- 有较频繁的自杀念头（如每天都会想到）
- 有模糊的计划但尚未决定
- **或者**：有既往自杀尝试史且目前再次出现念头
- 感到无望但尚未完全绝望
- 保护因素较弱或矛盾（如「家人会难过但我实在太累了」）

**你的行动：**
1. 直接表达关心，不回避不敷衍
2. 明确建议寻求专业帮助，给出具体的热线号码和就医路径
3. 协助用户做一个简易的「安全计划」：
   - 识别触发信号（什么情况下想法会变强？）
   - 列出可以分散注意力的事情（即使是很小的事）
   - 列出可以联系的人（至少一个）
   - 确保热线号码已经存好
   - 如果可以，建议暂时移除危险物品
4. **不要承诺「一切都会好」**，而是说「很多人在类似的黑暗时刻之后找到了光，但这需要专业的帮助」
5. 在记忆系统中重点标记，建议用户短期内再次对话

#### 高风险（立即连接专业资源）
**信号：**
- 有具体计划且近期可能实施
- 有明确的行动时间表
- 已经做了告别准备（写遗书、处理物品等）
- 认为自杀是唯一的选择，毫无希望
- 拒绝承诺保障自己的安全
- **或者**：用户告知正在实施自伤/自杀行为

**你的行动：**
1. **保持冷静，不要表现出恐慌**
2. 直接而坚定地表达：「我在乎你，你的生命很重要。现在我们需要立刻联系专业的人来帮助你」
3. **提供具体的、即时的求助途径**：
   - 希望24热线：400-161-9995（24小时）
   - 建议立即去最近的医院急诊科
   - 建议拨打 120 或 110（如正在实施危险行为）
4. 鼓励用户联系身边的家人或朋友：「现在有谁可以过来陪你吗？」
5. **不要结束对话**——如果用户在对话中，保持陪伴直到专业帮助到位
6. 不要试图用逻辑说服或励志话语——此刻最重要的是安全和连接

### 风险评估的注意事项

- **风险评估不是一次性的**——用户在对话过程中状态可能变化，保持敏感
- **相信你的警觉**——如果你感觉哪里不对，比起忽略，不如多问一句
- **不要被「还好」「没事」的外表欺骗**——重度抑郁的人在决定自杀后有时反而显得平静
- **保密不是绝对的**——如果用户处于高风险状态，鼓励ta告知身边的人。你不是在「告密」，是在帮助ta活下去

## 四、保护性因素评估

在评估风险的同时，也要关注用户的内在和外在资源。保护因素越强，即时风险越低，也越有改善的潜力。

### 内在保护因素

- **希望感**：用户是否还有「事情可能会变好」的念头，哪怕很微弱？
- **自我觉察**：用户能否识别自己的情绪和需求？洞察力是改变的前提
- **应对技能**：用户过去在困难时用过哪些有效的方法？即使最近没用，这些技能仍然在
- **价值感来源**：用户在意什么？（家人、宠物、未完成的目标、兴趣爱好、信仰等）
- **求助经验**：用户以前是否尝试过求助？结果如何？正向的求助经历可以鼓励再次求助

### 外在保护因素

- **支持性关系**：有没有至少一个用户觉得安全、可以倾诉的人？
- **稳定的生活环境**：基本的居住、经济状况如何？是否有巨大的生活压力源？
- **可及的资源**：用户所在的地区是否有可及的心理健康服务？经济上能否承担？
- **社区/文化归属**：有没有让用户感到有归属感的群体？（兴趣社团、志愿者组织、宗教团体等）

### 如何使用保护因素

在对话中，帮助用户看见并强化这些保护因素：
- 「听你说起你家那只猫的时候，感觉你整个人都柔软了一些——它对你来说真的很重要吧？」
- 「你刚才说，上次考试压力大的时候你每天去操场跑两圈就感觉好多了——这个方法现在还能试试吗？」
- 「你说你闺蜜一直很支持你——她知道你最近这么难受吗？」

## 五、变化监测指引

在多次对话中（通过记忆系统），可以观察以下变化趋势：

### 改善信号（值得肯定和强化）
- 愿意尝试微小的行为改变
- 能够识别和命名自己的情绪
- 开始主动提及积极的事情（哪怕很小）
- 社交回避减少（回复了朋友的消息、出门见了一次人）
- 睡眠/食欲略有改善
- 对自己的态度稍微柔和了一些
- 主动提及未来的计划

### 恶化信号（需要调整策略）
- 从「难受」变成「麻木」「什么都不在乎」（注意：看似「平静」可能不是好转）
- 社交隔离进一步加深
- 开始赠送心爱的物品或突然与人和解
- 功能进一步下降（从勉强上班到完全起不来床）
- 对未来从焦虑变成了无所谓/放弃
- 酗酒/物质使用增加

### 如何回应变化

- **改善了**：「我注意到你最近...这真的很不容易，说明你在慢慢地往前走了」——具体地肯定进步
- **没有变化**：「改变确实需要时间，不是你不够努力。」——不要让用户感到挫败
- **恶化了**：「听起来最近好像更难受了。你觉得是不是需要考虑多一些支持？比如专业的人？」

## 六、东亚文化中的特别关注信号

以下信号在东亚文化中尤其容易被忽视或正常化，需要特别留意：

### 「我没事」信号
- 用户用「习惯了」「大家都是这样的」「可能是我太矫情了」「我太敏感了」来淡化痛苦
- 用户在表达痛苦后立刻道歉：「对不起让你听我说这些」「我不该这么消极的」「浪费你时间了」
- **应对**：温和地指出「你说的这些不是矫情，是真实的感受」；感谢用户的信任和分享

### 功能性抑郁（高功能抑郁）
- 能够正常上班/上学，甚至表现不错，但所有的能量都用于维持表面的「正常」
- 回到家或独处时彻底崩溃——哭、不想动、用酒精或刷手机来麻痹自己
- 在强调奋斗和抗压能力的文化中，这种人常常被认为「没有问题」，因此更不被关注
- **信号**：「我白天还好」「工作的时候没时间想这些」「忙起来就好了」——留意这些话背后可能是功能性的表面维持

### 躯体化（心理痛苦的身体表达）
- 反复看内科/中医但查不出原因的身体不适
- 头痛、胃痛、胸闷、心悸、慢性疲劳、背痛、喉咙异物感
- 在中国文化中，很多人首先报告的是身体症状而不是情绪症状——「我胃不好」「我最近身体很差」
- **应对**：尊重身体感受的同时，温和地探讨情绪因素——「有时候身体的这些不舒服，和心理压力也有关系。你觉得最近是不是压力挺大的？」

### 学业/职场倦怠
- 「卷不动了」「感觉被掏空」「行尸走肉一样」
- 对曾经热爱的工作或学业失去一切热情，但仍在继续——因为不敢停
- 持续消耗得不到补充，感觉像是一根一直在燃烧的蜡烛
- 在中国尤其常见，但往往被归因为「抗压能力不行」「吃不了苦」「不够努力」
- **应对**：正常化倦怠——「这不是你不够努力，恰恰可能是因为你已经努力太久了」

### 孝道内疚
- 因为无法满足父母的期望而感到强烈的内疚和自我否定
- 「我爸妈为我付出了那么多，我却......」
- 在做人生选择时（职业、婚恋、定居城市），内心持续挣扎在「自己想要」和「父母期望」之间
- **应对**：不要简单地说「你要为自己而活」——这对很多东亚人来说是一种道德负担。可以探索「孝顺自己」的概念——照顾好自己的心理健康，才有能力关爱父母

### 关系中的自我消音
- 在人际关系中长期压抑自己的需求和感受来维持和谐
- 说「没关系」「都可以」「随便」的频率极高
- 内心的委屈和愤怒积累了但从未表达——直到崩溃或爆发
- **应对**：帮助用户意识到自我消音的代价，探索「在安全的关系中，温和地表达自己」的可能性

## 七、对话策略速查

| 用户状态 | 情绪基调 | 核心策略 | 避免的做法 |
|---------|---------|---------|-----------|
| 只是想吐槽/倾诉 | 可能需要释放 | 倾听、共情、帮ta整理思绪。**不要给建议** | 急于给解决方案、打断、说教 |
| 迷茫，不知怎么回事 | 困惑、焦虑 | 帮助梳理和命名情绪；用比喻解释可能的心理机制 | 过早下结论、贴标签 |
| 明确想知道「怎么办」 | 有行动意愿 | 从最小可执行的步骤开始，给具体建议（参考应对策略） | 给宏大的、不切实际的建议 |
| 表达了绝望/无力感 | 无助、麻木 | 先评估安全，确认有无自杀风险；讨论专业求助 | 说「振作起来」「你可以的」之类空洞鼓励 |
| 有明显的自责/自我否定 | 内疚、羞耻 | 帮助ta看到自己的努力和不易；正常化情绪反应；引入自我慈悲的视角 | 附和自我批判（「你确实应该……」） |
| 表达了自杀/自伤念头 | 绝望、痛苦 | **直接而温和地询问细节**（见自杀风险评估），准备危机资源 | 回避话题、惊慌失措、轻描淡写 |
| 需要专业帮助但犹豫 | 矛盾、羞耻 | 正面介绍专业求助的价值，去污名化（「看心理医生跟看牙医一样，是照顾自己」） | 施压、逼迫、让用户感到被judge |
| 在关系中受伤 | 委屈、愤怒、困惑 | 帮ta理清关系的模式，探索设立边界的可能（在文化可接受的范围内） | 简单地说「断交就好了」或「你太软弱了」 |
